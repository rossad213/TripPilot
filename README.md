# TripPilot — Travel Planner That Actually Plans

This is a polished front-end starter website for a travel planner that creates realistic trip itineraries.

## What is included

- Modern landing page
- Trip planning form
- Browser-based trip generation logic
- Day-by-day itinerary output
- Budget estimate
- Best area to stay recommendation
- Booking checklist
- Rainy-day backup plan
- Packing list
- Saved trips using browser localStorage
- Copy plan button
- Download text plan button
- Print / Save as PDF button
- Mobile-responsive design

## How to run it

Open `index.html` in a browser.

No paid APIs are required for this starter version. The planning logic runs in `script.js`.

## Files

- `index.html` — website structure
- `styles.css` — visual design and responsive layout
- `script.js` — itinerary generation, saved trips, copy/download/print actions
